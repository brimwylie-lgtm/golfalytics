# Arenalytics vs Golfalytics - What Changed

## Overview

This document shows exactly what was adapted from the Arenalytics site to create Golfalytics.

> "Privacy-compliant mobile foot traffic data that reveals how arena audiences behave..."

**Golfalytics:**

> "Privacy-compliant mobile foot traffic data that reveals how golfers behave..."
